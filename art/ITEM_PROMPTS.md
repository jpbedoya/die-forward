# Die Forward — Item Art Prompts

All prompts optimized for **DALL·E (ChatGPT)**. Use **512×512** for item icons (square).

---

## ⚠️ Shared Style Block

Every item uses this same visual language:

> Style: hyper-detailed dark fantasy digital painting, photorealistic, cinematic. Pure black background. Single dramatic amber/gold light source (torchlight) — strong contrast, deep shadows. Color palette: near-black, deep charcoal, amber gold, bone white, blood-rust red accents. Atmospheric fog, subtle glow. No bright colors, no neon, no cartoon style, no gradients. Square format 1024×1024.

**Item-specific rules:**
- Single object, centered, nothing near edges
- Object is partially wet or damp — it came from the crypt
- Strong readable silhouette at 64×64px
- No text, no UI, no labels

## Usage Notes

- All final item assets → `art/generated/items/`
- File naming: `item-{kebab-name}.webp` (e.g. `item-bone-charm.webp`, `item-void-salt.webp`)
- Target size: 512×512 WebP after conversion from DALL·E output
- Use `vert` to convert: `vert item-bone-charm.png item-bone-charm.webp`
- If output is too bright/colorful: add *"almost monochromatic, muted, desaturated"*
- If object reads as fantasy-cute rather than dark: add *"grimdark, decayed, aged, wet, dungeon-found"*
- Generate one first, confirm style matches existing creature art, then batch the rest
