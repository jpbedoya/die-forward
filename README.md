## âš”ï¸ Combat System

No HP trading ping-pong. Every choice is a **risk/reward tradeoff**. Read enemy intent, exploit weaknesses, gear up.

Enemies hit harder as you go deeper:
- **Tier 1**: Base damage (The Drowned, Pale Crawler)
