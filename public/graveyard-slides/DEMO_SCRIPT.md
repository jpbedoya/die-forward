## Key Moments to Nail

1. **Light a candle** early — shows social layer before your run starts
2. **DESCEND button** — not "Stake SOL", it's the main CTA on the title screen
3. **4 stake options in one row** — clean, no custom input
4. **Music playing** — audio audible, point to track name
5. **The death** — actually die, don't fake it. Animation + sound sells it.
6. **Final words** — type something memorable, it's on-chain forever
7. **Your death in Echoes** — close the loop, show it appeared in the feed
