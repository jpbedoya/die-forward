## Key Demo Moments to Nail

1. **Profile name** — Visible in header, shows Tapestry identity
2. **Light a candle** — Early in demo, shows social interaction before your run
3. **Stake confirmation** — Show the SOL leaving wallet
4. **Music playing** — Make sure audio is audible, point to track name
5. **On-chain indicator** — Show the MagicBlock status during run
6. **The death** — Don't fake it, actually die. The death animation + sound sells it.
7. **Final words** — Type something memorable, it'll be on-chain forever
8. **Leaderboard** — Show Ranks tab with top players
9. **Echo appearing** — Your death shows up in the feed. Full circle.

## Backup Plan

If something breaks mid-demo:
- Skip to slides, narrate what would happen
- "As you can see..." → show pre-recorded clip
- Have a second device ready with a run in progress
